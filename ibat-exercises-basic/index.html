HTML/CSS Exercises

  1. Complete these two tasks:
    • In CSS file, for h1 tag, change text-align and line-spacing.
    • In CSS file, for p tag, change color and line-height.
  2. Complete these two tasks:
    • In CSS file, add h2 tag, and its properties and values.
    • In CSS file, add h3 tag, and its properties and values.
  3. Complete these three tasks:
    • In CSS file, group h1, h2 and h3 for font-family.
    • In CSS file, group h1 and h2 for color.
    • In CSS file, for h3, add text-transform.
  4. Complete these two tasks:
    • In HTML file, add bold tag within second papagraph of text.
    • In HTML file, add italic tag within second papagraph of text.
  5. Complete these two tasks:
    • In HTML file, add new p.deckhead class tag for first pargraph of text.
    • In CSS file, add p.deckhead tag, and its properties and values.
  6. Complete these two tasks:
    • In HTML file, add new note-box class tag for first pargraph of text.
    • In CSS file, add .note-box tag, and its properties and values.
  7. Complete these two tasks:
    • In HTML file, add cat picture within img tag.
    • In HTML file, add dog picture within img tag.
  8. Complete these two tasks:
    • In HTML file, add class of img-center to the two img tags.
    • In CSS file, add new .img-center tag, and its properties and values.
  9. Complete these two tasks:
    • In HTML file, add three hyperliks using the href tag.
    • In CSS file, add new ul.toc and ul toc li classes, and their properties and values.
  10. Complete these two tasks:
    • In HTML file, enclose all content within body inside wrapper-content tag.
    • In CSS file, add new .wrapper-content tag, and its properties and values.
  11. Complete these three tasks:
    • In HTML file, use the href tag to add internal page hyperlinkks to table of contnes.
    • In CSS file, add new .wrapper-content tag, and its properties and values.
    • In HTML file, add unique IDs to create anchors for table of contents hyperlinks.
  12. Complete this task:
    • In HTML file, at bottom of page just before closing body tag add JavaScript file named smooth-scroll.js.
